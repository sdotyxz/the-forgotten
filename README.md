# The Forgotten (被遗忘之物)

一款为 Mini Jam 204: Café 制作的极简风寻物解谜游戏。

## 核心概念

> *"所见取决于所立之处。"*

旋转 1-bit 等距视角的咖啡馆场景，在桌椅和顾客之间寻找失落的物品。不同的角度会揭示不同的秘密。

## 游戏赛事

| 项目 | 内容 |
|------|------|
| **赛事** | Mini Jam 204: Café |
| **主题** | 咖啡馆 |
| **开发周期** | 72 小时 (2026年2月13-16日) |
| **引擎** | Godot 4.6 |
| **美术风格** | 1-bit 单色, 等距 3D |

## 操作说明

| 输入 | 动作 |
|------|------|
| 鼠标拖动 | 旋转场景 |
| 左键点击 | 收集物品 |
| R | 重置旋转 |
