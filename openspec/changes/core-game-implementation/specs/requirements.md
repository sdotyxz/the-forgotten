# 需求规格：核心游戏实现

## 1. 相机旋转控制

### 1.1 功能需求
- [ ] 鼠标拖动水平旋转场景
- [ ] 平滑旋转（插值）
- [ ] 可选：靠近 90° 倍数自动吸附
- [ ] R 键重置旋转

### 1.2 技术需求
- 围绕场景中心旋转
- 保持等距视角（-30°, 45°）
- 灵敏度可调

## 2. 1-bit 渲染

### 2.1 视觉效果
- [ ] 纯黑纯白输出
- [ ] 基于深度的边缘检测
- [ ] Bayer 抖动产生体积感
- [ ] 鼠标悬停反色高亮

### 2.2 性能需求
- 60 FPS
- 支持 1280x720 分辨率

## 3. 物品系统

### 3.1 物品属性
- 唯一名称
- 3D 位置
- 碰撞体
- 可见性（根据相机角度）

### 3.2 交互需求
- [ ] 鼠标悬停检测
- [ ] 点击收集
- [ ] 收集后消失/标记
- [ ] 收集反馈（音效+视觉）

## 4. UI 系统

### 4.1 HUD 元素
- [ ] 物品清单显示（顶部）
- [ ] 进度指示（X/5）
- [ ] 关卡名称
- [ ] 重置按钮

### 4.2 视觉风格
- 像素字体
- 1-bit 风格（黑白）
- 简洁边框

## 5. 验收标准
- [ ] 场景可流畅旋转
- [ ] 1-bit 渲染效果正确
- [ ] 物品可点击收集
- [ ] UI 显示正常
- [ ] 第一关可完整游玩
