# 变更提案：根据GDD实现游戏核心功能

## 变更名称
`implement-core-game`

## 提案原因
根据 GDD.md 设计文档，需要将 The Forgotten 游戏从概念转化为可玩的原型。当前 Godot 项目为空壳，需要实现核心功能。

## 目标
实现以下核心功能：
1. 1-bit 黑白等距渲染效果
2. 场景旋转控制（鼠标拖动）
3. 可收集物品系统
4. 简单的咖啡馆场景（第一关）
5. 基础 UI 显示

## 范围
- Godot 4.6 项目开发
- GDScript 脚本实现
- Shader 后处理效果
- 不涉及外部美术资源导入（使用程序化几何体）

## 预期产出
- 可运行的游戏原型
- 可旋转的 1-bit 场景
- 可点击收集的物品
- 关卡完成检测
