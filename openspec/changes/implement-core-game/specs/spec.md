# 需求规格

## 1. 功能需求

### 1.1 1-bit 渲染效果
- [ ] 实现全屏后处理 Shader
- [ ] 深度边缘检测（清晰轮廓）
- [ ] Bayer 抖动（体积感）
- [ ] 纯黑纯白输出

### 1.2 场景旋转控制
- [ ] 鼠标拖动水平旋转
- [ ] 平滑插值动画
- [ ] R 键重置视角
- [ ] 保持等距角度（-30°, 45°）

### 1.3 物品收集系统
- [ ] 可点击的物品
- [ ] 射线检测交互
- [ ] 收集后消失/标记
- [ ] 收集反馈

### 1.4 第一关场景
- [ ] 简单咖啡馆布局
- [ ] 3 个隐藏物品
- [ ] 基础家具（桌子、椅子等）

### 1.5 UI 系统
- [ ] 物品清单显示
- [ ] 进度指示
- [ ] 操作提示

## 2. 技术需求

### 2.1 Godot 配置
- 渲染器：Mobile
- 相机：正交投影
- 深度缓冲：启用

### 2.2 性能需求
- 60 FPS 稳定
- 启动时间 < 3 秒

## 3. 验收标准
- [ ] 场景可旋转
- [ ] 1-bit 效果正确
- [ ] 物品可收集
- [ ] 第一关可通关
